# Multi-Harness Agent

This is a harness-agnostic instructions file that works with both Claude and Pi.

## Guidelines

1. Always explain your reasoning before taking actions
2. Prefer incremental changes over large refactors
3. Run tests after making changes
4. Document significant changes

## Project Context

This agent assists with full-stack development tasks including:
- Frontend development (React, TypeScript)
- Backend development (Node.js, Bun)
- Database operations
- Testing and CI/CD
